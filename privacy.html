隐私协议

最后更新：2024年12月

概述

欢迎使用灵拓 Lynto（以下简称"我们"或"本应用"）。我们高度重视用户的隐私权， 并致力于保护您的个人信息安全。本隐私协议详细说明了我们如何收集、使用、存储和保护您的信息。

使用本应用即表示您同意本隐私协议的条款。如果您不同意本协议的任何部分， 请不要使用本应用。

1. 我们收集的信息

1.1 主动提供的信息

  • 聊天内容：您在应用中发送的消息和对话内容
  • 配置信息：您设置的 API 密钥、MCP 服务器配置等
  • 用户偏好：应用设置、语言偏好等个性化配置

1.2 自动收集的信息

  • 设备信息：设备型号、操作系统版本、应用版本
  • 使用统计：应用使用频率、功能使用情况（匿名收集）
  • 错误日志：应用崩溃报告和错误信息（不包含个人内容）

1.3 我们不收集的信息

  • 通讯录或其他应用的数据
  • 位置信息
  • 摄像头或麦克风数据（除非您主动使用语音功能）
  • 第三方应用的使用数据
  • macOS 相关数据（本应用仅支持 iPhone 和 iPad）

2. 信息使用方式

2.1 核心功能

  • AI 对话：您的聊天内容将发送到 DeepSeek AI 服务以生成回复
  • MCP 工具：配置的 MCP 服务器将处理相关工具调用
  • 个性化体验：根据您的设置提供定制化的用户体验

2.2 改进服务

  • 分析应用使用情况以改进功能
  • 诊断和修复技术问题
  • 开发新功能和服务

2.3 我们不会：

  • 出售您的个人信息给第三方
  • 将您的对话内容用于广告目的
  • 在未经授权的情况下分享您的个人数据

3. 信息存储和保护

3.1 本地存储

  • 配置数据：API 密钥和应用设置安全存储在您的设备上
  • 聊天历史：对话记录保存在您的设备本地数据库中
  • 缓存数据：临时文件和缓存数据存储在设备上

3.2 云端处理

  • AI 服务：对话内容通过加密连接发送到 DeepSeek AI
  • MCP 服务：工具调用数据发送到配置的 MCP 服务器
  • 临时存储：云端服务仅临时处理数据，不长期存储

3.3 安全措施

  • 使用行业标准的加密技术保护数据传输
  • API 密钥使用设备安全存储功能加密保存
  • 定期进行安全审查和更新
  • 实施访问控制和数据最小化原则

4. 第三方服务

4.1 DeepSeek AI

  • 您的对话内容将发送到 DeepSeek AI 进行处理
  • 请查阅 DeepSeek AI 的隐私政策了解其数据处理方式
  • 我们不控制第三方服务的数据处理方式

4.2 MCP 服务器

  • 您配置的 MCP 服务器将接收相关工具调用数据
  • 不同的 MCP 服务器可能有不同的隐私政策
  • 请在配置前了解相关服务的隐私条款

4.3 开源依赖

  • 本应用使用开源软件包，不涉及数据共享
  • 所有依赖项均经过安全审查
  • 详细的依赖列表可在源代码中查看

5. 您的控制权

5.1 数据管理

  • 查看数据：您可以随时查看应用中存储的所有数据
  • 修改数据：您可以编辑或删除聊天记录和配置
  • 导出数据：您可以导出聊天历史和配置文件
  • 删除数据：您可以清除所有本地数据

5.2 隐私设置

  • 选择是否启用使用统计收集
  • 控制错误报告的发送
  • 管理 API 密钥和服务配置
  • 设置数据保留期限

5.3 撤销同意

  • 您可以随时停止使用本应用
  • 卸载应用将删除所有本地数据
  • 您可以联系我们要求删除相关数据

6. 数据保留政策

6.1 本地数据

  • 聊天记录在您的设备上保留，直到您主动删除
  • 配置数据在应用卸载时自动删除
  • 缓存数据定期自动清理

6.2 处理数据

  • 发送到 AI 服务的数据仅用于生成回复
  • 工具调用数据在处理完成后不保留
  • 错误日志保留30天用于问题诊断

7. 儿童隐私保护

灵拓 Lynto 不专门面向13岁以下的儿童设计。我们不会有意收集13岁以下儿童的个人信息。 如果我们发现收集了儿童的个人信息，我们将立即删除相关数据。

如果您是家长或监护人，发现您的孩子向我们提供了个人信息， 请联系我们，我们将立即采取措施删除相关信息。

8. 国际数据传输

由于我们使用的第三方服务（如 DeepSeek AI）可能位于不同国家， 您的数据可能会被传输到您所在国家/地区之外的地方进行处理。

我们确保所有数据传输都使用适当的安全措施，并遵守适用的数据保护法律。 当数据传输到数据保护法律可能不同的国家时，我们会采取额外的保护措施。

9. 法律合规

在某些情况下，我们可能需要披露您的信息以遵守法律义务，例如：

  • 响应法院命令、传票或其他法律程序
  • 保护我们或他人的权利、财产或安全
  • 调查欺诈或其他非法活动
  • 遵守监管要求

在这种情况下，我们将仅披露必要的信息，并在法律允许的范围内通知您。

10. 隐私协议更新

我们可能会不时更新本隐私协议。重要变更包括：

  • 数据收集或使用方式的重大变化
  • 第三方服务的变更
  • 安全措施的更新
  • 法律要求的变更

当我们更新隐私协议时，我们将：

  • 更新本页面顶部的"最后更新"日期
  • 在应用中通知重要变更
  • 在必要时征求您的重新同意

12. 生效日期

本隐私协议自2024年12月起生效。如果您在此日期之后继续使用本应用， 即表示您接受本协议的条款。
